# vue2.ele.me
webpack2+vue2搭建饿了么，个人娱乐项目。
# 技术栈
webpack2 + vue2 + vuex + vue-router + ES6/7 + fetch + sass + svg
# 项目运行
git clone https://github.com/sk8ks/vue2.ele.me.git

cd vue2.ele.me

npm install

npm run dev (访问本地，运行后访问 http://localhost:8080)
# 项目功能
项目功能基本参照并模拟官方版本
* 首页
  1. 定位
  2. 搜索地址
  3. 地址附近商家列表
